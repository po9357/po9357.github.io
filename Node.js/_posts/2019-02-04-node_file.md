 노드에서 파일을 다루는 IO는 동기식, 비동기식 모두 지원한다.<br>
 기본적으로 노드는 비동기 처리를 선호하기 때문에 동기식 IO의 경우 Sync라는 키워드를 붙여 구분한다.

 파일을 다룰때 사용하는 모듈은 fs모듈을 사용한다.<br>

 ### Buffer객체

 여기서는 이 전 예제와 달리 **Buffer**객체를 사용한것을 볼 수 있다.<br>
